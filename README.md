# Krüger Consulting · Sales-Goldstandard-Audit Templates

5 ausfüllbare PDF-Templates für die regelmäßige Selbst-Analyse aller Sales-Calls nach Goldstandard.

| Modul | Datei | Rhythmus |
|---|---|---|
| Opening / Akquise | `Sales-Goldstandard_Opening.pdf` | alle 2 Wochen |
| Setting / Qualifikationsgespräch | `Sales-Goldstandard_Setting.pdf` | alle 2 Wochen |
| Closing / Verkaufsgespräch | `Sales-Goldstandard_Closing.pdf` | alle 2 Wochen |
| Kundenberatung (KB) Audit | `Sales-Goldstandard_KB.pdf` | monatlich |
| KB · Meeting-Vorbereitung | `Sales-Goldstandard_KB-Vorbereitung.pdf` | vor jedem KB-Meeting |

PDFs enthalten klickbare Radio-Buttons (Bewertung 1–5) und ausfüllbare Textfelder. Speichern via `Cmd+S`.
